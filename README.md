✨ Özellikler
Kullanıcı Girişi: Güvenli ve hızlı giriş ekranı.
Keşfet (Explore): Kimlerin, nerede, ne giydiğini keşfet; yeni trendleri ve popüler gönderileri yakından takip et tarzını keşfet öneri ve fikir al.
Gelişmiş Arama: Aradığın tarzda kombin fikirlerine anında ulaş, sana en uygun parçaları kolayca bul.
Etkileşim: Beğendiğin kombinlere yorum at, tarzını beğendiğin kullanıcılarla etkileşime geç.
Profil Yönetimi: Kişisel stilini sergileyebileceğin özelleştirilebilir kullanıcı paneli.

🛠 Kullanılan Teknolojiler
Bu proje aşağıdaki teknolojiler kullanılarak geliştirilmiştir:
Dil: C#
Framework: Windows Forms
Veritabanı: Microsoft SQL Server
Arayüz Tasarımı: C# ile dinamik olarak oluşturulmuş form ve kontrol yapıları
Geliştirme Ortamı: Visual Studio 2022
